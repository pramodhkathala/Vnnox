# vnnox-rendered-html
Automated script + instructions to capture the rendered HTML from:
https://au.vnnox.com/care/#/schemeConf/generate
and publish it on GitHub Pages.

## Steps

### 1) Generate rendered HTML locally
1. Install Node and npm if not present.
2. Clone this repo or put files into a folder.
3. Install puppeteer:
   ```bash
   npm init -y
